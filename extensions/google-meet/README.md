# @grokbot/google-meet

Official Google Meet participant plugin for GrokBot.

This plugin registers the `google_meet` tool so agents can join Google Meet calls through supported Chrome or Twilio transports.

## Install

```bash
grokbot plugins install @grokbot/google-meet
```

Restart the Gateway after installing or updating the plugin.

## Configure

Enable the plugin and follow the Google Meet docs for browser profile, transport, and call-join setup:

- https://docs.grokbot.ai/plugins/google-meet

## Package

- Plugin id: `google-meet`
- Tool: `google_meet`
- Package: `@grokbot/google-meet`
- Minimum GrokBot host: `2026.4.20`
