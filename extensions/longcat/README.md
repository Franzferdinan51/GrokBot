# GrokBot LongCat Provider

Official GrokBot provider plugin for LongCat-2.0 through LongCat's hosted
OpenAI-compatible API.

Install from GrokBot:

```bash
grokbot plugins install @grokbot/longcat-provider
grokbot gateway restart
```

Configure `LONGCAT_API_KEY`, then select `longcat/LongCat-2.0`.
