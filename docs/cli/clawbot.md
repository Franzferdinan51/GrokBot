# `grokbot clawbot`

Legacy alias namespace kept for backward compatibility. It registers the same QR command as the top-level CLI, so `grokbot clawbot qr` accepts every [`grokbot qr`](/cli/qr) flag.

## Migration

Prefer the modern top-level command:

- `grokbot clawbot qr` -> `grokbot qr`
