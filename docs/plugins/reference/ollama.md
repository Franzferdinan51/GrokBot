# Ollama plugin

Adds Ollama, Ollama Cloud model provider support to GrokBot.

## Distribution

- Package: `@grokbot/ollama-provider`
- Install route: included in GrokBot

## Surface

providers: `ollama`, `ollama-cloud`; contracts: `memoryEmbeddingProviders`, `tools`, `webSearchProviders`
